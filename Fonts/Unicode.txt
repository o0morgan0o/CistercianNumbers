Unicode fonts

It is possible to input cistercian numerals or glyphs using a font such the ConScript Unicode Registry: 
https://unifoundry.com/pub/unifont/unifont-14.0.01/font-builds/unifont_csur-14.0.01.ttf

Or using the Standard Galactic Alphabet included here.

These will not show up correctly unless you have one of the aforementioned fonts installed.
